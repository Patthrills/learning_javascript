String length: Prints out the length of a string
String slice(): slices out a part of a string
String substring(): Slices out the parts of a string just like .slice(), the diffrence is it accepts 2 parameters
String substr(): Almost like .substring(), it accept 2 parameters too. the difference is that the second parameter starts counting from where the parameter is cut off
String replace(): replaces a text in a string (it is case sensitive but you can and "/i" to bypass the text)
String replaceAll(): replaces all text in a sctring
String toUpperCase(): changes texts in a string to upper case.
String toLowerCase(): changes texts in a string to lower case.
String concat(): concatenates 2 strings together.
String trim(): cleans up spaces around a string.
String trimStart(): cleans up spaces at the begining a string.
String trimEnd(): cleans up spaces at the end of a string around a string.
String padStart pads a string with another string from start
String padEnd(): pads a string with another string from end
String charAt(): Prints out the character of a particular position
String charCodeAt(): returns unicode of a particular character
String split() converts a string to an array.
